## Descripción

**SecLap** es un sistema web para la gestión y control de préstamo de equipos de cómputo en una institución educativa. Permite administrar el inventario de laptops, registrar préstamos y dar seguimiento al estado de los equipos.

---

## Problemática

En muchas instituciones educativas, el control de equipos de cómputo se realiza de manera manual, lo que provoca pérdida de información, falta de control sobre los préstamos y dificultades para conocer la disponibilidad real de los equipos.

Este proyecto surge para digitalizar y mejorar el control del inventario y los préstamos de equipos.

---

## Solución Propuesta

Se desarrolló una plataforma web que permite:

- Registro y administración de equipos de cómputo
- Control de préstamos y devoluciones
- Gestión de usuarios
- Consulta de disponibilidad de equipos
- Historial de préstamos

La solución centraliza la información para mejorar el control y la trazabilidad de los equipos.

---

## Arquitectura

El sistema está basado en una arquitectura cliente-servidor:

- Frontend web desarrollado con HTML, CSS y PHP
- Backend en PHP
- Base de datos relacional en PostgreSQL
- Lógica de negocio integrada en el servidor

---

## Desarrollo del Proyecto

El desarrollo inició con el análisis del problema de control de inventario en el entorno educativo.

Posteriormente se diseñó la base de datos en PostgreSQL, definiendo entidades como usuarios, equipos y préstamos.

Se implementó el sistema en PHP, desarrollando formularios, validaciones y lógica de negocio para la gestión de préstamos.

Finalmente se realizaron pruebas funcionales para validar el registro y control de equipos.
